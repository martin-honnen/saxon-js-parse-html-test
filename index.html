Some tests to see whether the saxon:parse-html function can be implemented for client-side XSLT 3 with Saxon-JS 2 using JavaScript

Saxon Java PE and EE implement an extension function saxon:parse-html to parse a string of HTML markup into an XDM document node. Here I try to implement a replacement for that function using new DOMParser().parseFromString(html, 'text/html') called using ixsl:eval in XSLT 3 run in Saxon-JS 2.

  • Successful use of DOMParser from XSLT 3 run by Saxon-JS 2
